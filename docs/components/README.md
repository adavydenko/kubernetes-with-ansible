# Компоненты кластера

В этом разделе собраны материалы по отдельным компонентам кластера Kubernetes:

- балансировщик нагрузки MetalLB;
- подсистема хранилища и Local Storage;
- мониторинг и мониторинг NVIDIA;
- поддержка GPU и Multi-GPU.

Используйте эти главы как справочник и дополнение к основному учебному пособию.

## Роли Ansible и связи между компонентами

Автоматизация в этом репозитории строится вокруг набора ролей: сначала поднимается control plane и сеть, затем подключаются MetalLB, хранилище, мониторинг и демо-приложение. Ниже — схема зависимостей и типичного порядка применения ролей (см. также [руководство по развертыванию](../deployment/DEPLOYMENT_GUIDE.md)).
