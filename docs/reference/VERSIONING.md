# Версии

Этот документ фиксирует правила версионирования учебной документации, матрицы совместимости и историю изменений.

### Текущий релиз документации

- **Текущая версия:** `1.1.0`
- **Дата релиза:** `2026-04-17`
- **Статус:** `Stable`
- **О версии:** версия `1.1.0` объединяет актуальные главы по MetalLB, Local Storage, Monitoring, GPU/Multi-GPU и расширенные учебные материалы.

#### SemVer

Версионирование учебного пособия ведется по схеме SemVer (`MAJOR.MINOR.PATCH`):

- **MAJOR**: несовместимые изменения структуры или требований (например, крупная переработка разделов и учебных треков).
- **MINOR**: новые главы, разделы, практические блоки и справочные материалы без существенных изменений общей структуры учебного пособия.
- **PATCH**: исправления ссылок, примеров, терминологии, опечаток и неточностей.

#### Как обновлять этот документ

При каждом релизе документации:

1. Обновляйте блок **Текущий релиз документации**.
2. Актуализируйте все таблицы **Матрица совместимости**.
3. Добавляйте новый раздел в **История изменений (Changelog)**.
4. Проверяйте ссылки на версионный документ в обзорных разделах.

## Матрица совместимости

- `✅ Полная`: сценарии подтверждены, документация актуальна.
- `⚠️ Частичная`: часть материала применима, но требует дополнительной проверки.
- `❌ Устарела`: версия не поддерживается и не рекомендуется для новых внедрений.
